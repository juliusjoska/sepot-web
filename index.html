Funkce

Soukromí bez kompromisů

Navrženo od základu s důrazem na bezpečnost. Žádné zadní vrátka, žádné výmluvy.

E2E šifrování

Zprávy jsou šifrovány na vašem zařízení. Nikdo jiný je nemůže přečíst - ani my.

Žádná metadata

Nesbíráme nic. Ani kdo s kým píše, ani kdy, ani jak často. Úplné soukromí.

Open source

Kód je otevřený a auditovatelný. Můžete si ověřit, že děláme to, co říkáme.

Lokální šifrování

Data jsou šifrována i na vašem zařízení. Při ztrátě telefonu zůstávají v bezpečí.

P2P architektura

Server slouží pouze k navázání spojení. Zprávy jdou přímo mezi vámi.

Skupinové chaty

Bezpečné skupinové konverzace s plným E2E šifrováním pro všechny účastníky.

Bezpečnost

Prověřená kryptografie

Používáme stejné algoritmy jako Signal a Threema. Ověřené, auditované, doporučované bezpečnostními experty.

  • Perfect Forward Secrecy - kompromitace klíče neodhalí historii
  • Zero-knowledge architektura - server neví nic o obsahu
  • Žádné zadní vrátka - ani pod nařízením soudu
  • Auditovatelný kód - kdokoli může ověřit bezpečnost
Použité algoritmy
Výměna klíčů X25519
Šifrování zpráv XSalsa20-Poly1305
Digitální podpisy Ed25519
Derivace klíčů Argon2id
Knihovna libsodium
O projektu

Proč Šepot?

Šepot vznikl jako experiment v oblasti soukromé komunikace. Chtěli jsme vytvořit messenger, kde máte plnou kontrolu nad svými daty. Projekt je open source a aktivně se vyvíjí.
